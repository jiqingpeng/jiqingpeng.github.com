下载node

Node下载地址

第一步：安装cordova

执行命令

cnpm install cordova

第二步：新建cordova项目

执行命令

cordova create cordovaApp com.cordova.testapp
cd cordovaApp
cordova platform add android

第三步：打包APk

执行命令

cordova build android
